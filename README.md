# CredHex-App
 Secure digital vault for storing and accessing your certificates online.

CredHex is an innovative digital vault designed to simplify and secure the way you manage your academic and professional certificates. Built with modern web technologies, CredHex allows users to effortlessly register, log in, and upload important documents in PDF format. With the backing of Supabase for real-time storage and authentication, users can confidently store their credentials in a centralized, easily accessible location.
